# Express Users CRUD API

A RESTful backend API built with **Node.js and Express.js** that implements core **CRUD operations** following clean API design principles.

---

## Overview

This project demonstrates practical backend fundamentals such as routing, request validation and in-memory data management using Express.

---

## Key Features

- RESTful API design  
- Create, read, update, and delete user records  
- Route-based request handling  
- Input validation and error handling  
- Clean and readable Express code  

---

## Tech Stack

- Node.js  
- Express.js  
- JavaScript (ES Modules)  

---

## Project Intent

Built as a **backend fundamentals project** to strengthen understanding of how APIs work internally before moving to databases and authentication.

---

## Limitations

- Uses in-memory mock data  
- Data resets on server restart  
- Not intended for production use
